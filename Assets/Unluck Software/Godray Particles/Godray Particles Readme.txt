Adjusting Particles
To scale particles use the included particle scale tool: https://assetstore.unity.com/packages/--/simple-particle-scaler-9703
Change to Shape property to adjust the box the particles emits from.
Increase or decrease the effect by changing the Emission property or the color of the particles.

Compatible with Unity 2021.3 up to new versions of Unity(6+)
